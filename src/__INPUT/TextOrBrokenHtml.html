Si uno de nuestros clientes no puede actualizar ActivaERP, se deberá intentar restablecer la caché de resolución de nombres para que resuelva correctamente activaerp.com hacia xxx.xx.xx.xxx.

 

¿Que se debe hacer?:

1.       Reiniciar ordenador.

2.       Ejectuar en cmd > ipconfig /flushdns Seguramente después hará falta resetear el ordenador

3.       Modificar el archivo hosts del ordenador (sólo hacer si no hay más remedio)

a.       Editar el archivo C:\Windows\System32\drivers\etc\hosts añadiendo dos lineas al final del mismo:

                                                               i.      144.76.91.234 www.activaerp.com

                                                             ii.      144.76.91.234 activaerp.com
